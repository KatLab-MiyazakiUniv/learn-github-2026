# learn-github-2026
Git, GitHub, Google Testについて勉強するリポジトリ

## Google Test の実行について

### 1. Cmake をインストール
- **Linux(WSL)の場合**

    以下のコマンドを実行
    ```
    $ sudo apt update
    $ sudo apt install cmake
    ```

- **macOS**
[参考記事](https://zenn.dev/choozoo/articles/eb3de5da27d364)

### 2. テストを実行
以下のコマンドを実行

```
$ make
```
